# DTrade Terminal 🚀

![DTrade Logo](assets/images/logo.jpeg)

**DTrade** is a professional-grade, high-performance trading terminal built with Flutter. Designed for precision and speed, it offers a sophisticated interface for monitoring markets, analyzing AI-driven signals, and executing trades with a premium "terminal" aesthetic.

---

## ✨ Key Features

### 📊 Advanced Trading Interface
- **Real-time Market Tickers**: Live tracking of BTC/USD, ETH/USD, XAU/USD, SPX, NASDAQ, and major Forex pairs.
- **Integrated Charting**: Professional-grade visualization of market trends and price action.
- **Execution Engine**: Seamless BUY/SELL execution with customizable Lot sizes, Stop Loss, and Take Profit.

### 🤖 Market Intelligence
- **AI-Driven Alerts**: Real-time notifications for high-probability setups and market shifts.
- **Signals Hub**: Curated trading signals across all asset classes (CRYPTO, STOCKS, FOREX).
- **Intelligence View**: Deep-dive analytics and sentiment analysis for smarter decision-making.

### 🧪 Paper Trading Simulator
- **Risk-Free Practice**: Test strategies with virtual funds using real-time market prices.
- **Account Management**: Initialize your virtual balance and track your progression.
- **Trade History**: Comprehensive logging of closed positions and PnL performance.

## 🛠️ Technology Stack

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (>= 3.5.0)
- Android Studio / VS Code with Flutter extension
- Firebase project configured (Google Services files required for Android/iOS)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YourUsername/dtrade.git
   cd dtrade
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Configure Firebase:**
   - Add your `google-services.json` to `android/app/`.
   - Add your `GoogleService-Info.plist` to `ios/Runner/`.

4. **Run the app:**
   ```bash
   flutter run
   ```
